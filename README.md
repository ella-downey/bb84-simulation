# bb84-simulation
This notebook walks through the quantum cryptography BB84 protocol, allowing users to adjust encrypted key lengths, visualize the decryption process, and analyze statistical effects of eavesdropping.

This project aims to simulate the BB84 Protocol using Python to allow for an exploration of the impact bit-flip error, phase-error, depolarization, and eavesdropping have on transmitted keys. The code (attached independently) follows the encoding, decoding, eavesdropping, and error-checking processes described in the BB84 Protocol section. It allows users to visualize the protocol on a single encrypted bit-string, following its development through each step.
